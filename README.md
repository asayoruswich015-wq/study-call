# スタディコール

エビングハウス忘却曲線×AI学習管理アプリ

## GitHub Pagesで公開する手順

1. GitHubアカウントを作成（https://github.com）
2. 右上「+」→「New repository」をクリック
3. Repository name に「study-call」と入力
4. 「Public」を選択して「Create repository」
5. 「uploading an existing file」をクリック
6. このフォルダの中身（index.html, manifest.json, sw.js, icon.svg）を全部ドラッグ＆ドロップ
7. 「Commit changes」をクリック
8. 「Settings」→「Pages」→「Branch: main」を選択して「Save」
9. 数分後に https://あなたのユーザー名.github.io/study-call/ でアクセス可能！

## ホーム画面に追加（Android）

1. ChromeでURLを開く
2. 右上「⋮」→「ホーム画面に追加」
3. 完了！アプリとして起動できます
